# Uniting for Sustainable Progress - SDG 17

## üé® UI Style Guide

This project follows a specific visual identity designed to balance urgency with professionalism.

### 1. Color Palette
| Purpose | Hex Code | Visual Sample |
| :--- | :--- | :--- |
| **Primary Accent** | `#f28e52` | (Orange) Used for News Cards and Action Buttons |
| **Secondary Background** | `#2d6a6a` | (Teal) Used for Footer and Newsletter Background |
| **Text (Inverse)** | `#ffffff` | (White) Used for text on dark backgrounds |
| **Page Background** | `#e0e0e0` | (Light Grey) Used for main content separation |

### 2. Typography
* **Headings:** Serif font (Georgia/Times New Roman) for a trustworthy, editorial feel.
* **Body Text:** Sans-serif or Serif for readability across all device sizes.
* **Font Weights:** Bold (700) for headers and button labels; Regular (400) for descriptions.

### 3. Components
* **Action Buttons:** Rounded pill-shape (`border-radius: 25px`) with high-contrast orange background.
* **Input Fields:** Minimalist design with `border-bottom` only, creating a clean, modern aesthetic.
* **Cards:** Solid color blocks with white-bordered images to make visuals pop.
